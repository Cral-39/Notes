# CSS Basics
***
### 一.基础认知
#### 1.1 CSS的介绍
+ css的作用是给页面中的HTML标签设置样式
#### 1.2 CSS语法规则
+ CSS写在style标签中，style标签一般写在head标签里面，title标签下面
+ 一个实例：
```HTML
<!-- 前面省略 -->
<title>Ducument</title>
<style>
    p{
        /* 在下面设置一些属性 */
        color:xxx;
        front-size:xxxpx;
        backgrond-color:xxx;
        width:xxxpx;
        height:xxxpx;
    }
</style>
<!-- 省略 -->
<body>
    <!-- 标签内的文字会按照上面的属性变化 -->
    <p>这是一个p标签</p>
</body>
```
